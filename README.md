# HTML-CSS Website

This project is a static website created for a high school certification project.

## Description
The website was built by adapting and customizing a template, while using HTML and CSS to organize content, connect multiple pages, and style the visual layout.

## What I Worked On
- Customized the structure and content of the pages
- Connected multiple HTML pages through internal navigation
- Edited and styled sections using CSS
- Added images, GIFs, buttons, and background elements
- Adjusted layout and page presentation

## Technologies Used
- HTML
- CSS
- W3.CSS

## Note
This is not a fully original design from scratch, since it was developed starting from a template. However, the project was customized and modified as part of the learning process and school project requirements.
